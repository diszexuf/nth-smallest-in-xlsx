# N-ое минимальное число в XLSX

Сервис находит **N-ое наименьшее число** в первом столбце XLSX-файла

Используется **QuickSelect (O(n) в среднем)** [без библиотечных функций]

---

## Сборка и запуск

1. Клонировать репозиторий
```bash
git clone <repo-url>
cd minimum-number
```
2. Сборка и генерация openapi

```bash
./gradlew clean build
```
3. Запуск приложения
```bash
./gradlew bootRun
```

## Описание метода

- алгоритм QuickSelect с выбором опорного элемента по медиане из 3-х значений 
- средняя временная сложность O(n), из-за медианы из 3-х практически уходит вероятность ухудшения до O(n^2)
- O(n^2) возможен при отсортированном массиве и плохом выборе опорного элемента без медианышт
---
## Работа алгоритма: 
1) выбираем медиану
2) разбиваем массив на две части: меньше и больше опорного элемента
3) продолжаем поиск только в части, где находится нужный индекс
   
(таким образом получается частичная сортировка)
