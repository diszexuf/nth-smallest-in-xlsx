# N-ое минимальное число в XLSX

Сервис находит **N-ое наименьшее число** в первом столбце XLSX-файла

## Стек технологий

- **Java** : 21
- **Spring Boot** : 3.5.7 
- **Apache POI** : 5.4.1 
- **OpenAPI Generator** : 7.15.0
- **Springdoc OpenAPI UI** : 2.8.13
- **Lombok** : 1.18.34+
- **JUnit 5** : 5.10+

## Требования

- **Java 21**
- **Gradle 8.5+** (или `gradlew`)
- **XLSX-файл** с числами в **первом столбце**

## Сборка и запуск

1. Клонировать репозиторий
```bash
git clone https://github.com/diszexuf/nth-smallest-in-xlsx.git
cd nth-smallest-in-xlsx
```

2. Сборка и генерация openapi

```bash
./gradlew clean build
```

4. Запуск приложения
```bash
./gradlew bootRun
```

## Использование

После сборки и запуска swagger будет доступен по адресу 

```http://localhost:8080/swagger-ui/index.html```

## Описание метода

- алгоритм QuickSelect с выбором опорного элемента по медиане из 3-х значений 
- средняя временная сложность O(n), из-за медианы из 3-х практически уходит вероятность ухудшения до O(n^2)
- O(n^2) возможен при отсортированном массиве и плохом выборе опорного элемента без медианы
